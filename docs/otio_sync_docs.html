Overview

This documentation describes the OTIO-based synchronized review messaging protocol. All events inherit from the base SyncEvent class and are serialized as OpenTimelineIO objects.

Note: This documentation was automatically generated from the Python schema definitions. Examples are created from actual OTIO class instances defined in the configuration file.
