PartialAnnotation

schema: Annotation.1event: PARTIAL

Mid-stroke partial annotation (visual preview, not persisted).

FieldTypeDescription
clip_guidstrSync GUID of the clip being annotated.
framefloat0-indexed clip-local frame number.
fpsfloatFrame rate used to interpret 'frame'.
eventslistSerialized SyncEvent dicts for the in-progress stroke.

Example

{
  "clip_guid": "clip-guid-001",
  "frame": 12.0,
  "fps": 24.0,
  "events": [
    {
      "OTIO_SCHEMA": "PaintStart.1",
      "uuid": "stroke-001"
    }
  ]
}

schema: LiveSession.1event: STATE_SNAPSHOT
